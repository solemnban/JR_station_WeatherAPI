💡 機械学習データセット構築に関するテクニカルメモ

本ダッシュボードは、収集された data/train_logs.csv を直接非同期にパースして描画するインタラクティブなフロントエンドです。

GitHub Pages上に展開することで、手元のローカル環境を立ち上げることなく、常にデータの蓄積量、アノテーション状態（遅延原因のテキスト分布）、特徴量のスケーリングをビジュアル確認でき、予測モデル（LightGBM / LSTM等）の構築プロセスを強力に支援します。
